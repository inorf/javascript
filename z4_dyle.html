1. Kiedy powstał JavaScript (podaj datę).
    Odpowiedź: JavaScript został stworzony w 1995 roku.

  2. Dlaczego jest to język skryptowy?
    Odpowiedź: JavaScript jest językiem skryptowym, ponieważ wymaga kompilacji do kodu maszynowego w trakcie działania programu.

  3. Gdzie jest wykonywany JS (klient lub serwer)?
    Odpowiedź: JavaScript jest wykonywany po stronie klienta, czyli w przeglądarce internetowej użytkownika.

  4. Dlaczego JavaScript jest bezpiecznym narzędziem?
    Odpowiedź: JavaScript jest uważany za narzędzie o ograniczonych uprawnieniach dostępu do zasobów komputera, co przyczynia się do jego bezpieczeństwa.

  5. Czy JavaScript jest językiem obiektowym, jeśli tak, to dlaczego?
    Odpowiedź: Tak, JavaScript jest językiem programowania obiektowego, ponieważ wykorzystuje definicje obiektów, metod i zdarzeń do opisu właściwości obiektów.

  6. Wymień zalety JavaScript:
    • Zaleta 1: JavaScript jest stosunkowo łatwy do opanowania, co pozwala zacząć używać go już po kilku minutach nauki.
    • Zaleta 2: Ze względu na swoją ogromną popularność w Internecie można znaleźć wiele praktycznych przykładów witryn wykorzystujących różne funkcje JavaScriptu.
    • Zaleta 3: JavaScript pozwala na odciążenie serwerów i ograniczenie ilości zbędnych danych przesyłanych pomiędzy przeglądarką klienta a serwerem.

  7. Framework to:
    Odpowiedź: Framework to rodzaj programu służącego do tworzenia innych programów w języku JavaScript.

  8. Do aplikacji mobilnych można użyć frameworka:
    Odpowiedź: Do tworzenia aplikacji mobilnych w języku JavaScript można wykorzystać framework o nazwie "React Native".

  9. Aplikacje responsywne to:
    Odpowiedź: Aplikacje responsywne to takie, które potrafią dostosować swój wygląd i zachowanie do bieżącego rozmiaru ekranu, co pozwala na lepsze działanie na różnych urządzeniach.

  10. Przepisz poniższą linię kodu i podaj wyjaśnienie, jak działa ten fragment kodu:
    <input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">
    Odpowiedź: Ten przycisk otwiera nowe okno przeglądarki po kliknięciu, ale nie zawiera żadnej konkretnej strony do otwarcia.

  11. Przepisz poniższą linię kodu i podaj wyjaśnienie, jak działa ten fragment kodu:
    window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
    Odpowiedź: Ten kod otwiera nowe okno przeglądarki o określonych właściwościach. Opcje, takie jak "toolbar=no," "directories=no," "menubar=no," kontrolują wygląd okna, a pozostałe opcje ustawiają jego rozmiar i położenie na ekranie.

  12. Opisz następującą instrukcję window.close()
    Odpowiedź: Instrukcja window.close() służy do zamknięcia aktualnego okna przeglądarki lub zakładki, w której jest wykonywana. Pozwala użytkownikowi zakończyć przeglądanie strony lub zamknąć pop-upowe okno, jeśli jest ono aktywne.
