    2. Czym jest funkcja w JS, omów przy użyciu 4 podpunktów?

  1. Jest to zbiór zgrupowanych instrukcji, które możemy uruchamiać poprzez podanie ich nazwy
  2. Każda funkcja po wywołaniu wykonuje swój wewnętrzny kod
  3. Funkcja może mieć argumenty, które wchodzą do wnętrza funkcji i są tam przetwarzane np. suma_kry(a, b)
  4. Może(nie zawsze) zwrócić obliczoną wartość zapisaną po słowie kluczowym return

    3. Omów metody getElementsByName, getElementById, getElementByTagName (może być w postaci tabeli)

  1. getElementById("id ") -> dostęp do wszystkich elementów o konkretnej wartości atrybutu id
  2. getElementByTagName("znacznik ") -> dostęp do wszystkich elementów o typie znacznika, np. P lub DIV
  3. getElementByName("nazwa") -> dostęp do wszystkich elementów o konkretnej wartości atrybutu

    4. Omów właściwości innerHTML innerText className. (każdą osobno)

  1. innerHTML – jest to właściwość, która umożliwia odczyt, zmodyfikowanie, wstawienie, albo usunięcie kodu HTML z danego elementu HTML np. w tabeli
  2. innerText – umożliwia określenie ciągu znaków który zostanie umieszczony w elemencie
  3. className – umożliwia określenie klasy CSS użytej do prezentacji elementu
